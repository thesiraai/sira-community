## Discourse

SIRA Community is a fork of [Discourse](https://github.com/discourse/discourse), an open-source community platform created and maintained by [Civilized Discourse Construction Kit, Inc.](https://www.discourse.org/).

### About Discourse

Discourse is a 100% open-source community platform that has been battle-tested for over a decade. It provides a modern, feature-rich foundation for building online communities with:

- Discussion forums and topics
- Real-time chat
- User management and moderation
- Extensive plugin and theme system
- Comprehensive API and integration options

## SIRA Community

SIRA Community builds upon the Discourse foundation to provide integrated community functionality for the SIRA AI ecosystem. We maintain compatibility with Discourse's core features while adding SIRA-specific integrations and customizations.

### SIRA Community Team

The SIRA AI team has adapted Discourse for integration with the SIRA AI application, including:
- Docker deployment configuration
- SIRA API integration
- Custom branding and theming
- Integration documentation

## How to Contribute

If you're interested in contributing to SIRA Community:

1. Review our [CONTRIBUTING.md](CONTRIBUTING.md) guide
2. Check existing issues and pull requests
3. Follow our code of conduct
4. Submit improvements and bug fixes

For contributions that would benefit the broader Discourse community, we encourage you to also contribute upstream to the [official Discourse repository](https://github.com/discourse/discourse).
